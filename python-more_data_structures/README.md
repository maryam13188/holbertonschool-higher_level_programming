# Python - More Data Structures

This project contains Python scripts for various data structure operations including sets, dictionaries, lists, and Roman numeral conversion.

## Tasks Description

### 0. Squared Simple
**File:** `0-square_matrix_simple.py`  
Compute the square of all integers in a 2D matrix.

### 1. Search and Replace
**File:** `1-search_replace.py`  
Replace all occurrences of an element in a list.

### 2. Unique Addition
**File:** `2-uniq_add.py`  
Add all unique integers in a list (only once for each integer).

### 3. Present in Both
**File:** `3-common_elements.py`  
Return a set of common elements in two sets.

### 4. Only Differents
**File:** `4-only_diff_elements.py`  
Return a set of all elements present in only one set.

### 5. Number of Keys
**File:** `5-number_keys.py`  
Return the number of keys in a dictionary.

### 6. Print Sorted Dictionary
**File:** `6-print_sorted_dictionary.py`  
Print a dictionary with keys sorted alphabetically.

### 7. Update Dictionary
**File:** `7-update_dictionary.py`  
Replace or add key/value in a dictionary.

### 8. Simple Delete by Key
**File:** `8-simple_delete.py`  
Delete a key from a dictionary.

### 9. Multiply by 2
**File:** `9-multiply_by_2.py`  
Return a new dictionary with all values multiplied by 2.

### 10. Best Score
**File:** `10-best_score.py`  
Return the key with the biggest integer value in a dictionary.

### 11. Multiply by Using Map
**File:** `11-multiply_list_map.py`  
Return a list with all values multiplied by a number using `map()`.

### 12. Roman to Integer
**File:** `12-roman_to_int.py`  
Convert a Roman numeral to an integer.

## Learning Objectives

- Understanding Python sets and their common methods
- Working with dictionaries and their operations
- When to use sets vs lists vs dictionaries
- Using lambda functions, map, and list comprehensions
- Iterating over sets and dictionaries
- Converting Roman numerals to integers

## Requirements

- Python 3.8.5
- Ubuntu 20.04 LTS
- All files must end with a new line
- First line of all files: `#!/usr/bin/python3`
- Code must follow pycodestyle (version 2.7.*)
- All files must be executable

## How to Use

1. Clone the repository
2. Make files executable: `chmod +x *.py`
3. Run individual scripts: `./[script_name].py`

## Example Usage

```bash
# Task 0: Square matrix
./0-main.py

# Task 12: Roman to integer
./12-main.py
```
