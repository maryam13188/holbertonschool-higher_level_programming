# JavaScript DOM Manipulation

## Description
This project contains JavaScript scripts for manipulating the DOM, handling events, and fetching data from APIs.

## Files

| File | Description |
|------|-------------|
| `0-script.js` | Changes header text color to red |
| `1-script.js` | Changes header color to red when clicking on `#red_header` |
| `2-script.js` | Adds class `red` to header when clicking on `#red_header` |
| `3-script.js` | Toggles between classes `red` and `green` on header when clicking on `#toggle_header` |
| `4-script.js` | Adds `<li>Item</li>` to `<ul class="my_list">` when clicking on `#add_item` |
| `5-script.js` | Changes header text to "New Header!!!" when clicking on `#update_header` |
| `6-script.js` | Fetches Star Wars character name (Leia Organa) from API and displays it in `#character` |
| `7-script.js` | Fetches Star Wars movies titles from API and displays them in `<ul id="list_movies">` |
| `8-script.js` | Fetches French translation of "hello" ("Salut") from API and displays it in `#hello` |

## Technologies Used
- JavaScript (ES6+)
- Fetch API
- DOM Manipulation
- Async/Await

## Requirements
- Chrome browser (version 57.0 or later)
- No use of `var` (use `const` or `let`)
- Code follows semistandard style
- HTML does not reload for each action
