## Python - Input/Output & Serialization
Overview

This project focuses on file handling and data persistence in Python. It covers reading and writing text files, working with JSON serialization/deserialization, handling command-line arguments, and implementing a basic algorithm (Pascal’s Triangle).

## Key Topics

File I/O using the with statement

JSON serialization and deserialization

Object-to-dictionary conversion

Data persistence to disk

Command-line argument handling

Basic algorithm implementation

## Tasks

The project includes 13 tasks (0–12) covering:

Reading, writing, and appending files

Converting Python objects to/from JSON

Saving and loading objects from files

Serializing class instances

Rebuilding objects from stored data

Implementing Pascal’s Triangle
