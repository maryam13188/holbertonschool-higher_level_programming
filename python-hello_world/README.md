
# Python - Hello, World

This project is an introduction to Python programming.
It focuses on learning the basics of the Python language,
including printing output, using variables, strings,
indexing, slicing, and following the Python style guide.

## Learning Objectives
At the end of this project, you should be able to explain:

- How to use the Python interpreter
- How to print text and variables using `print`
- How to work with strings in Python
- What indexing and slicing are
- What the official Python coding style is
- How to check Python code using `pycodestyle`

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- Python version: **Python 3.8**
- All files must end with a new line
- The first line of all files must be exactly:

- All files must be executable
- Code must follow `pycodestyle` (version 2.7.*)

## Files
| File | Description |
|------|-------------|
| `2-print.py` | Prints a specific sentence |
| `3-print_number.py` | Prints an integer using f-strings |
| `4-print_float.py` | Prints a float with 2 decimal places |
| `5-print_string.py` | Prints a string multiple times and slices it |
| `6-concat.py` | Concatenates strings |
| `7-edges.py` | String slicing (copy, cut, paste) |
| `8-concat_edges.py` | Creates a new sentence from existing strings |
| `9-easter_egg.py` | Prints The Zen of Python |
